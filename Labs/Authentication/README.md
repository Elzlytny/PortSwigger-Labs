# Authentication

This directory contains my **Authentication Guide**, **Authentication Playbook**, technical writeups, professional vulnerability reports, and supporting screenshots for the **Authentication** labs from PortSwigger Web Security Academy.

---

## About This Category

Authentication vulnerabilities occur when an application fails to properly verify the identity of users before granting access.

Weak authentication mechanisms can allow attackers to enumerate valid usernames, brute-force passwords, bypass authentication controls, compromise user accounts, or even gain unauthorized access.

This repository documents not only the solution for each lab, but also the authentication concepts, testing methodology, technical writeups, professional vulnerability reports, and playbook developed throughout the learning process.

In addition to solving the labs, I documented each finding with:

- Authentication Guide
- Authentication Playbook
- Professional Writeups
- Professional Penetration Testing Reports
