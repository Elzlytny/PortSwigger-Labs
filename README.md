# PortSwigger Web Security Academy Labs

A practical portfolio documenting my journey through the PortSwigger Web Security Academy.

## About

This repository contains my solutions, writeups, and professional reports for PortSwigger Web Security Academy labs. The primary goal is not only to solve labs, but also to build a structured methodology for Web Application Penetration Testing.

## Objectives

* Improve Web Penetration Testing skills.
* Practice identifying and exploiting real-world web vulnerabilities.
* Develop a repeatable web application testing methodology.
* Write professional vulnerability reports.
* Build a public portfolio demonstrating practical cybersecurity skills.
